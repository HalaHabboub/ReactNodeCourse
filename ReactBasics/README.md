## Study Order

For beginners, it's recommended to study in this order:

1. **Start with Bundlers and Transpilers** - Understand the tools
2. **Move to SPA** - Understand the architecture
3. **Then Virtual DOM** - Understand React's core optimization
4. **Finally Components** - Understand how to build with React

---

## Quick Reference

| Concept | Key Point |
|---------|-----------|
| **Bundlers** | Combine multiple JS files into one |
| **Transpilers** | Convert modern syntax to browser-compatible code |
| **SPA** | One page, dynamic updates, no reloads |
| **Virtual DOM** | Fast, in-memory representation of real DOM |
| **Function Components** | Simple functions that return JSX |
| **Class Components** | ES6 classes with render() method |
| **Hooks** | Give function components state and lifecycle features |
